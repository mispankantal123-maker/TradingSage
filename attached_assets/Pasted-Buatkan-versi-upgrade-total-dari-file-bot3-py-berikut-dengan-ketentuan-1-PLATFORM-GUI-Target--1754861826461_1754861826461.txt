Buatkan versi upgrade total dari file bot3.py berikut, dengan ketentuan:

1. PLATFORM & GUI

Target Windows desktop, bukan web dan bukan Flask.

Gunakan tkinter untuk GUI, pastikan tampilannya rapi, responsif, dan tidak freeze.

Semua tombol, input, dropdown harus berfungsi penuh, tidak ada dummy atau placeholder.

GUI harus punya:

Dropdown pilihan strategi trading

Dropdown list symbol dari MT5 + input manual symbol (user bisa ketik)

Auto-detect symbol aktif jika user tidak memilih manual

Input lot, auto-lot toggle, risk %, TP & SL dengan pilihan unit:

pips

price

percent

money

Tombol Start/Stop bot (mengontrol loop autotrading)

Tombol Connect/Disconnect MT5 (dengan status koneksi real-time)

Log area real-time

Status Trading: Running / Stopped

2. KONEKSI MT5

Gunakan metode koneksi dari bot3.py (karena sudah terbukti work auto connect).

Pastikan auto reconnect jika koneksi terputus.

Saat konek, tampilkan info di log:

Status koneksi (Berhasil/Gagal)

Nama akun & balance

Saat disconnect, tampilkan pesan jelas di log.

Validasi simbol:

Harus bisa validasi untuk semua jenis pair: forex, crypto, gold, index.

Jika simbol tidak ditemukan, coba variasi umum (BTCUSDm, BTCUSD, BTCUSD.a, dll.).

Jika masih gagal, tampilkan pesan error di GUI dan log.

3. SISTEM TRADING

Ambil semua strategi yang ada di bot3.py:

Scalping

Intraday

Arbitrage

High Frequency Trading

Setiap strategi punya TP/SL sendiri (per-strategy config).

Trading loop berjalan di thread terpisah agar GUI tidak freeze.

Auto order BUY/SELL sesuai sinyal strategi.

Manual BUY/SELL tombol di GUI harus berfungsi penuh.

Hitung TP/SL berdasarkan unit yang dipilih (pips, price, %, money).

Pastikan semua order dari bot terkirim ke MT5 dan return result ke log.

4. STABILITAS & KELENGKAPAN

Tidak ada tombol dummy atau fungsi kosong.

Tidak ada file dummy atau placeholder.

Semua input dan dropdown terhubung ke fungsi yang nyata.

Bot harus bisa langsung jalan di Windows setelah python main.py dijalankan.

Konfigurasi tersimpan otomatis (config.json) dan dimuat saat start.

Log tersimpan di file + tampil di GUI.

Tangani semua error MT5 agar bot tidak crash.

5. PENINGKATAN DARI bot3.py

GUI lebih rapi dan lengkap.

Koneksi MT5 auto dan lebih informatif.

Symbol input dan autodetect lebih fleksibel.

TP/SL support semua unit (pips, %, price, money).

Strategi berjalan lancar tanpa freeze.

Semua fitur bot3.py tetap ada tapi ditingkatkan.

6. OUTPUT FILES

main.py → entry point

gui.py → tkinter GUI

trading.py → koneksi MT5, strategi, eksekusi order

utils.py → config, logging, helper

requirements.txt

Pastikan tidak ada fungsi yang kosong, tidak ada tombol hiasan, dan semua fitur langsung terpakai untuk auto trading live di MT5.
